# React Todo List App

This is a simple React application built for practice purposes. It demonstrates core React concepts such as component-based architecture, state management, and props.

## 📝 Features

- Add new tasks
- Delete tasks
- Update task details
- Mark tasks as completed

## 🧱 Components

The app is structured into three main components:

- **Home** – Serves as the main landing component.
- **TaskList** – Displays the list of all tasks and add a new task.
- **Task** – Represents individual task items with options to edit, delete, or mark as complete.
